# Password generator
## Description
A Python project that generates secure random passwords using letters, numbers, and special characters.

## Features
- User-defined password length
- Secure password generation
- Strong password creation
## Technologies Used
- Python
- string module
- secrets module
## How to Run
1. Open the Python file.
2. Run the program.
3. Enter the password length.
4. View the generated password.
## Sample Output
```text
Enter password length: 10

Generated Password:
A#7kP9@mQ2
```

- Python modules
- User input handling
- String manipulation
- Secure password generation
